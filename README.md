# Start from none
- edit package.json
- include name, version, main
- use webpack, babel to render react component

# Get started
- Clone the project
- run npm install
- modify code
  - If modified react, run npm run webpack to rebuild

# Build App
- run npm run webpack-pro
  - build as production mode 
- run npm run build
- the output will under dist file

# How to use
- In dist/URPlanner-win32-x64 file, double click kelvin icon
- All src placed in {file}/src folder
  - Include Mission.xml and Mate.pdf
- Updated mission.xml need to place in src folder with same name
